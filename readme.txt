== Description ==

This plugin is useful to convert your existing classic block/editor content to gutenberg blocks.


== Why we built this plugin?==

Currently it is manual process to convert existing page into the gutenberg blocks system. But "Convert Classic Editor to Gutenberg Blocks" is convert automatically existing page to gutenberg block.

== Key Features ==

"Convert Classic Editor to Gutenberg Blocks" is convert many HTML elements like Heading Tag(H1 to H6), Paragraph, Div, Image, Span, List, WP Shortcode etc to gutenberg block.

**Note**

This plugin require <a href="https://wordpress.org/plugins/multipurpose-block/" target="_blank">Multipurpose Gutenberg Block</a> plugin to work correctly.
